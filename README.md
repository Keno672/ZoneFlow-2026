# ZoneFlow v1 HTML App

Static GitHub-ready app prototype for the ZoneFlow 3-card system.

- Free: Intel + News
- Paid: Battlefield
- Scan upload populates Intel
- Chart / zone brief populates Battlefield
- News upload populates News

Open admin.html, paste a one-page brief, save it, then return to index.html. The app stores data in browser localStorage for this v1 prototype.
